## 🎵 项目简介  
一个专注于 **美观界面设计** 的本地音乐播放器，支持通过文件夹管理音乐库。虽然功能简洁，但通过 **QML/Qt Quick** 实现了优雅的视觉交互，支持主题颜色切换。  
> **注意**：本播放器 **不支持在线音乐搜索**，仅提供本地文件播放和基础功能。

---

## 🌟 核心功能  
- **本地音乐管理**：通过文件夹快速添加/浏览本地音乐库  
- **播放控制增强**：  
  - 随机播放  
  - 循环模式  
  - 播放顺序调整  
- **主题系统**：  
  - 提供 **5种预设主题**（深色/浅色/霓虹/暮光/极简）  
  - 支持自定义主题颜色（通过配置文件修改）  

 

---

## 🛠️ 技术栈  
| 技术          | 作用描述                          |
|---------------|-----------------------------------|
| **Qt Quick**  | 实现动态界面与动画效果            |
| **QML**       | 构建响应式用户界面                |
| **CMake**     | 项目构建与依赖管理                |
| **JSON配置**  | 主题颜色与播放设置持久化存储      |
